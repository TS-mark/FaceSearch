###百度Ai人脸识别项目Demo程序
------
##注：本程序创建日期为2019.5.15，使用人脸识别V3版本
将项目文件图片转存到FaceRecongnition.java的FilePath下，直接编译运行程序，项目maven远程仓库地址为[https://maven.aliyun.com/mvn/search](https://maven.aliyun.com/mvn/search)
Demo项目token自动获取，直至申请的该AI权限过期为止。
输出数据为
```

```
则表示Demo程序正常运行
